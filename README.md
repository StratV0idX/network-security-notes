## 🔐 Password Strength Checker Enhancement

This feature improves the password checker with:

- GUI interface using Tkinter  
- Password strength scoring (0–5)  
- Suggestions for stronger passwords  
- Detection of common passwords using a wordlist  

### 📂 Wordlist

A lightweight wordlist (~815KB) is included in:

assets/wordlist.txt

This is used to detect commonly used passwords and simulate real-world dictionary attacks.

> Note: The wordlist is for demonstration purposes and can be replaced with larger datasets if needed.
